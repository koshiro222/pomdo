# Phase 09 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | Tailwind CSS v4.2.1 (shadcnスタイルインポート済み、components.jsonなし) |
| Preset | Cozy Forest（ダークベース） |
| Component library | lucide-react（アイコン）、Framer Motion（アニメーション） |
| Icon library | lucide-react v0.575.0 |
| Font | Manrope（Google Fonts） |

**既存パターン**: `.glass` クラスで統一されたカードスタイル

---

## Spacing Scale

Declared values（既存コードベースから検出、4の倍数）:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | アイコンギャップ |
| sm | 8px | コンパクトな要素間隔 |
| md | 16px | デフォルトの要素間隔（gap-4） |
| lg | 24px | セクションパディング |
| xl | 32px | レイアウトギャップ |

**例外**:
- `gap-1` (4px) — 週次チャートの棒間隔
- `max-w-[20px]` — 週次チャートの棒幅

## Typography

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body | 14px (text-sm) | 400 (regular) | 1.5 |
| Label | 12px (text-xs) | 700 (bold) | 1.5 |
| Heading | 24px (text-2xl) | 700 (bold) | 1.2 |
| Display | なし | - | - |

**既存パターン**:
- カードヘッダー: `text-xs uppercase tracking-widest text-cf-subtext font-bold mb-4`
- 数値表示: `text-2xl font-bold`
- サブラベル: `text-xs text-cf-subtext`

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #121814 (--df-bg-surface) | 背景サーフェス |
| Secondary (30%) | #1a211c (--df-bg-elevated) | カード、モーダル |
| Accent (10%) | #22c55e (--df-accent-primary) | タブ選択状態、グラフ色、完了アイコン |
| Destructive | #ef4444 (--df-accent-danger) | エラー状態 |

**Accent予約対象**:
- タブボタンの選択状態（Today/Week/Month）
- グラフの棒と折れ線
- アイコンコンテナの背景（`bg-cf-primary/20`）
- Todayの棒グラフ

**セカンダリ色**:
- #3b82f6 (青) — フォーカス/アクティブ状態
- #f59e0b (琥珀色) — 休憩状態

## Component Patterns

### Stats Card Structure

```
.glass（rounded-3xl p-6 h-full flex flex-col）
├── ヘッダー（text-xs uppercase tracking-widest text-cf-subtext font-bold mb-4）
├── 本日の統計（w-10 h-10 rounded-xl bg-cf-primary/20 アイコン）
├── タブ切り替え（flex gap-2 mb-4）
└── タブコンテンツ（Today/Week/Month）
```

### タブパターン

```tsx
<button className={`px-4 py-2 rounded-lg ${
  activeTab === 'today' ? 'bg-cf-primary text-white' : 'bg-white/10'
}`}>
  Today
</button>
```

### グラフパターン（Recharts ComposedChart）

```tsx
<ResponsiveContainer width="100%" height={200}>
  <ComposedChart data={weeklyData}>
    <XAxis dataKey="date" />
    <YAxis yAxisId="left" />
    <YAxis yAxisId="right" orientation="right" />
    <Tooltip />
    <Bar yAxisId="left" dataKey="sessions" fill="#22c55e" name="セッション数" />
    <Line yAxisId="right" type="monotone" dataKey="cumulativeMinutes" stroke="#22c55e" name="累積時間(分)" />
  </ComposedChart>
</ResponsiveContainer>
```

### 統計表示パターン

```tsx
<div className="flex items-center gap-3 mb-3">
  <div className="w-10 h-10 rounded-xl bg-cf-primary/20 flex items-center justify-center">
    <Clock className="w-5 h-5 text-cf-primary" />
  </div>
  <div>
    <p className="text-2xl font-bold text-cf-text">{value}</p>
    <p className="text-xs text-cf-subtext">{label}</p>
  </div>
</div>
```

## Animation Patterns

### Framer Motionタブ切り替え

```tsx
import { motion, AnimatePresence } from 'framer-motion'

<motion.div
  layout="position"
  initial={{ opacity: 0 }}
  animate={{ opacity: 1 }}
  transition={{ duration: 0.2 }}
>
  {tabContent}
</motion.div>
```

### グラフアニメーション

- Rechartsデフォルトアニメーション有効
- 週次チャート: `initial={{ opacity: 0, y: 10 }}, animate={{ opacity: 1, y: 0 }}, transition={{ delay: index * 0.05 }}`

---

## State Patterns

### ローディング状態

```tsx
{loading && (
  <div className="absolute inset-0 flex items-center justify-center bg-black/20 rounded-3xl">
    <div className="w-8 h-8 border-2 border-cf-primary border-t-transparent rounded-full animate-spin" />
  </div>
)}
```

### 空状態

```tsx
{completedSessions.length === 0 && !loading && (
  <div className="flex flex-col items-center justify-center h-full text-cf-subtext">
    <BarChart3 className="w-16 h-16 mb-4 opacity-50" />
    <p className="text-center">
      セッションがありません<br />
      ポモドーロを始めて記録を残しましょう
    </p>
  </div>
)}
```
