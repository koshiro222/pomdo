## Interaction Contract

このフェーズ固有のインタラクション仕様（アニメーション実装の指針）。

### LAYOUT-01: 仕切り線の配置

```
変更前:
  .bento-card
    └── div.border-b.pb-4.mb-4  ← ヘッダー全体を囲む border-b

変更後:
  .bento-card
    ├── div (ヘッダー — border-bなし)
    │   ├── "Tasks" + バッジ
    │   └── Current Task highlight
    ├── div.border-b.border-white/10.my-0  ← 独立した仕切り線
    ├── フィルタータブ（仕切り線の下）
    └── スクロールエリア
```

- 仕切り線: `border-b border-white/10` の `<div>` 要素
- 仕切り線の上下マージン: Current Taskセクションと`mb-3`、フィルタータブと`mt-3`

### LAYOUT-02: TodoInputの配置

- TodoInputをヘッダー内 (`div.border-b` の内部) から、スクロールエリア内の最下部へ移動
- スクロールエリア: `flex flex-col gap-3` の最後の子要素としてTodoInputを配置
- TodoInputはリストと一緒にスクロールする（カード底部固定なし）
- 表示状態: 常にボタン形状（"+ Add a new task..."）が見える

### ANIM-01: 既存アイテムのレイアウトアニメーション

- `TodoItem` の `motion.div` に `layout` prop を追加（`layout="position"` ではなく `layout` 単体）
- 新タスク追加時に既存アイテムが下方向へスムーズにスライドしてスペースを確保
- `AnimatePresence mode="popLayout"` は変更不要（既存）

### ANIM-02: 新アイテムの出現アニメーション

`slideInVariants`（x方向スライド）を以下の展開アニメーションに置き換える:

```typescript
// src/lib/animation.ts に追加する新バリアント
export const expandInVariants: Variants = {
  hidden: {
    opacity: 0,
    height: 0,
    marginBottom: 0,
  },
  visible: {
    opacity: 1,
    height: "auto",
    transition: {
      height: { duration: 0.25, ease: [0, 0, 0.2, 1] },
      opacity: { duration: 0.2, delay: 0.05, ease: [0, 0, 0.2, 1] },
    },
  },
  exit: {
    opacity: 0,
    height: 0,
    transition: {
      height: { duration: 0.2, ease: [0.4, 0, 1, 1] },
      opacity: { duration: 0.15, ease: [0.4, 0, 1, 1] },
    },
  },
}
```

- `TodoItem` の `variants` を `expandInVariants` に変更
- `initial={isNew ? 'hidden' : false}` — 既存タスクはアニメーションなし（リロード時に全アイテムがアニメーションしないよう `false`）
- `animate="visible"` は変更なし
