update: protectedProcedure
  .input(updateTodoSchema)
  .mutation(async ({ ctx, input }) => {
    // 存在チェック
    const existing = await db
      .select()
      .from(ctx.schema.todos)
      .where(eq(ctx.schema.todos.id, input.id))
      .limit(1)

    if (!existing || existing.length === 0) {
      throw new TRPCError({ code: 'NOT_FOUND', message: 'Todo not found' })
    }

    // 部分フィールド更新
    const updateData: any = {}
    if (input.title !== undefined) updateData.title = input.title
    if (input.completed !== undefined) updateData.completed = input.completed

    const updated = await db
      .update(ctx.schema.todos)
      .set(updateData)
      .where(eq(ctx.schema.todos.id, input.id))
      .returning()

    return updated[0]
  })

2. bgmRouterにupdate mutation追加（createの後）：
```typescript
update: adminProcedure
  .input(updateBgmTrackSchema)
  .mutation(async ({ ctx, input }) => {
    const { db, schema } = ctx

    // 存在チェック
    const existing = await db
      .select()
      .from(schema.bgmTracks)
      .where(eq(schema.bgmTracks.id, input.id))
      .limit(1)

    if (!existing || existing.length === 0) {
      throw new TRPCError({
        code: 'NOT_FOUND',
        message: 'トラックが見つかりません',
      })
    }

    // 部分フィールド更新
    const updateData: Record<string, unknown> = {}
    if (input.title !== undefined) updateData.title = input.title
    if (input.artist !== undefined) updateData.artist = input.artist
    if (input.color !== undefined) updateData.color = input.color
    if (input.tier !== undefined) updateData.tier = input.tier

    const [updated] = await db
      .update(schema.bgmTracks)
      .set(updateData)
      .where(eq(schema.bgmTracks.id, input.id))
      .returning()

    return updated
  })
```

実装の注意点：
- todos.tsのupdateパターンを参考に、存在チェック→部分更新の手順を実装
- Record<string, unknown>型でupdateDataを宣言（TypeScriptエラー回避）
- undefinedチェックで「フィールドが指定されたか」を判定
- updatedAtはdefaultNow()により自動更新される（DBレベルのトリガー）
- エラーメッセージは日本語

CONTEXT.mdの決定事項を遵守：更新可能フィールドはtitle, artist, color, tierのみ。filenameは変更不可（ファイル置換はスコープ外）。
