# BGM 音源の配置方法

以下の CC0 / Royalty-Free 音源を MP3 形式でダウンロードし、このディレクトリに配置してください。

## 推奨ファイル名

| ファイル名 | 説明 |
|-----------|------|
| `lofi-01.mp3` | Lo-Fi Study 01 |
| `lofi-02.mp3` | Lo-Fi Study 02 |

## 無料音源の入手先

- [Pixabay Music](https://pixabay.com/music/search/lofi/) — CC0、商用利用可
- [Free Music Archive - HoliznaCC0](https://freemusicarchive.org/music/holiznacc0/) — CC0、パブリックドメイン
- [Chosic](https://www.chosic.com/free-music/lofi/) — Royalty-Free

## 注意

`.mp3` ファイルは `.gitignore` によりリポジトリに含まれません。
各自でダウンロードして配置してください。
