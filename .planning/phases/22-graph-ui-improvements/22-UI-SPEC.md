## Component Specifications

### StatsCard グラフエリア (Weekタブ)

**構造:**
```
.bento-card (親コンテナ)
  └─ .flex-1.flex.flex-col.min-h-0 (グラフエリア)
      └─ ResponsiveContainer (width="100%" height={aspect-based})
          └─ ComposedChart
              ├─ XAxis (tickLine={false}, axisLine={false})
              ├─ YAxis left (tickLine={false}, axisLine={false})
              ├─ YAxis right (tickLine={false}, axisLine={false})
              ├─ Tooltip
              ├─ Bar (yAxisId="left", fill="#22c55e")
              └─ Line (yAxisId="right", stroke="#22c55e", strokeWidth={2})
```

**スタイル仕様:**

| 要素 | プロパティ | 値 | ソース |
|------|-----------|-----|--------|
| ResponsiveContainer | height | `{Math.max(200, parentWidth * 0.4)}` (動的aspect) | Claude裁量 |
| XAxis | tickLine | `false` | CONTEXT.md |
| XAxis | axisLine | `false` | CONTEXT.md |
| XAxis | tickFormatter | `(value) => value.slice(0, 3)` (3文字表記維持) | CONTEXT.md |
| XAxis | stroke | `#9ca3af` (text-cf-subtext) | CONTEXT.md |
| YAxis | tickLine | `false` | CONTEXT.md |
| YAxis | axisLine | `false` | CONTEXT.md |
| YAxis | stroke | `#9ca3af` (text-cf-subtext) | CONTEXT.md |
| Bar | fill | `#22c55e` (df-accent-primary) | 既存コード |
| Line | stroke | `#22c55e` (df-accent-primary) | 既存コード |
| Line | strokeWidth | `2` | 既存コード |
| ComposedChart | accessibilityLayer | `true` | CONTEXT.md |

**レスポンシブ挙動:**
- デスクトップ: 高さは親コンテナの幅に基づいて動的計算（aspect比維持）
- モバイル: デスクトップと同じ高さロジック、幅に応じて縮小（ResponsiveContainer既存挙動）

## Interaction States

### タブ切り替え (Today/Week/Month)

| State | Background | Text | Border |
|-------|------------|------|--------|
| Active | `bg-cf-primary` (#22c55e) | `text-white` | なし |
| Inactive | `bg-white/10` | `text-cf-subtext` (#9ca3af) | なし |

既存実装を維持:
```css
.bento-card:hover {
  border-color: rgba(255, 255, 255, 0.15);
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3);
}
```
