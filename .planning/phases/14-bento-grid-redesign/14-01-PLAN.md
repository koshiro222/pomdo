src/App.tsx に以下の変更を行う。カード内部コンテンツには一切触れない。

**1. 外側コンテナ（L151）の overflow 変更:**
```tsx
// 変更前: className="font-display bg-cf-background text-cf-text min-h-screen relative overflow-hidden"
// 変更後:
className="font-display bg-cf-background text-cf-text min-h-screen relative overflow-x-hidden lg:overflow-hidden"
```
`overflow-hidden` → `overflow-x-hidden lg:overflow-hidden` に変更。モバイルで縦スクロールを有効にしつつ、横スクロールは防止。デスクトップでは従来通り overflow-hidden。

**2. Flexコンテナ（L163）のブレイクポイント変更:**
```tsx
// 変更前: className="relative z-10 flex flex-col min-h-screen sm:h-screen"
// 変更後:
className="relative z-10 flex flex-col min-h-screen lg:h-screen"
```
`sm:h-screen` → `lg:h-screen` に変更。lg未満ではviewport固定を解除しスクロール可能にする。

**3. グリッドdiv（L168）のクラス変更:**
```tsx
// 変更前: className="h-full grid grid-cols-1 sm:grid-cols-6 lg:grid-cols-12 gap-4"
// 変更後:
className="h-full grid grid-cols-1 lg:grid-cols-3 gap-4"
```
12列システムを廃止して3カラム均等に切り替え。

**4. Timerカード（L170-189）のクラス変更:**
```tsx
// 変更前: className="bento-card sm:col-span-4 sm:row-span-2 lg:col-span-8 lg:row-span-2"
// 変更後:
className="bento-card"
```
col-span/row-span指定を全て削除。custom={0} は維持。

**5. CurrentTaskCardのDOM削除:**
L192-201のCurrentTaskCard `motion.div` ブロック全体を削除する。
L13の `import CurrentTaskCard from './components/tasks/CurrentTaskCard'` も削除する。
（Phase 15でTodoListに統合予定。Phase 14ではDOMから削除が最も安全 — 3カラムシステムに4番目のセルが入るとレイアウトが崩れるため。）

**6. TodoListカードの位置変更とクラス修正:**
現行のTodoList（L228-237）をBGMPlayerの前に移動し、グリッドの2番目のセルにする。

```tsx
// 変更前: className="bento-card sm:col-span-6 lg:col-span-12 lg:row-span-1"
// 変更後:
className="bento-card"
```
col-span指定を全て削除。custom={1} に変更（Timerの次）。

**7. BGM+Statsラッパー追加:**
BGMPlayerとStatsCardをplain divラッパーで包む。ラッパーには `.bento-card` も `motion.div` も付けない。
```tsx
{/* カラム3: BGM + Stats ラッパー（plain div） */}
<div className="h-full flex flex-col gap-4">
  <motion.div
    className="bento-card flex-1"
    variants={fadeInUpVariants}
    initial="hidden"
    animate="visible"
    custom={2}
    layout="position"
  >
    <BgmPlayer />
  </motion.div>
  <motion.div
    className="bento-card flex-1"
    variants={fadeInUpVariants}
    initial="hidden"
    animate="visible"
    custom={3}
    layout="position"
  >
    <StatsCard />
  </motion.div>
</div>
```
BGMPlayerのcustom={2}、StatsCardのcustom={3}。両方に `flex-1` を追加して均等50%高さ分割。

**最終的なグリッド内のDOM順序:**
1. Timer (motion.div, custom=0)
2. TodoList (motion.div, custom=1)
3. BGM+Statsラッパー (plain div)
   - BGMPlayer (motion.div, custom=2, flex-1)
   - StatsCard (motion.div, custom=3, flex-1)

**変更しないもの:**
- main タグの `className="flex-1 p-4 min-h-0"` — そのまま維持
- fadeInUpVariants の import — そのまま維持
