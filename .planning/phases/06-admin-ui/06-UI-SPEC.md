## Interaction Patterns

### Modal Open/Close
- Trigger: Header admin button (Settings icon)
- Animation: Framer Motion `AnimatePresence` + `motion.div`
- Duration: 200ms, `easeOut`
- Close on: Background click, Escape key, X button

### List Item Actions
- Edit/Delete buttons: Hidden by default, show on hover (`opacity-0 group-hover:opacity-100`)
- Hover: Border color change (`hover:border-cf-primary/30`)
- Tap: Framer Motion `tapAnimation` (scale: 0.95)

### Form Validation
- File size: Real-time check on file select (max 10MB)
- File type: MP3 only (`audio/mpeg`)
- Error display: Inline text below file input

### Loading States
- Add/Edit: Button disabled + text change (「追加中...」, 「更新中...」)
- Delete: Dialog button disabled + text change (「削除中...」)
- List fetch: Simple loading text (「Loading...」)

### Toast Notifications
- Success: Green background, message
- Error: Red background, error message
- Position: Top-right or top-center (existing `useUiStore` pattern)

## Visual Specifications

### Admin Modal
- Width: `max-w-2xl` (672px)
- Padding: `p-6` (24px)
- Border radius: `rounded-xl` (16px)
- Background: `widget` (glassmorphism)
- Header: Flex row, title (text-lg font-bold), close button (X icon)

### Track List
- Container: `flex flex-col gap-2 max-h-96 overflow-y-auto`
- Empty state: `text-cf-subtext text-center py-8`
- Section header: Flex row, title (text-sm font-bold), add button

### Track Item
- Container: `group flex items-center gap-3 px-4 py-3 bg-white/5 rounded-xl border border-white/5 hover:border-cf-primary/30 transition-colors`
- Color preview: `w-4 h-4 rounded-full` with `backgroundColor`
- Tier badge: `text-xs bg-cf-primary/20 text-cf-primary px-2 py-0.5 rounded`
- Action buttons: `opacity-0 group-hover:opacity-100 transition-opacity`

### Form Elements
- Input fields: `rounded-lg border border-cf-border bg-cf-surface px-3 py-2 text-sm text-cf-text outline-none focus:border-cf-primary transition-colors`
- Labels: `text-xs text-cf-subtext`
- Buttons: `bg-cf-primary hover:bg-cf-primary/90 text-white px-4 py-2 rounded-lg transition-colors cursor-pointer disabled:opacity-50`

## Accessibility Requirements

- Keyboard navigation: Tab through form inputs, Escape to close modal
- Focus management: Return focus to trigger after modal close
- ARIA labels: Add to icon-only buttons (Edit, Delete, Close)
- Color contrast: All text meets WCAG AA (4.5:1 for body text)
- Screen reader: Track item structure with semantic HTML
