      }
      body {

      }

    変更後:
    ```css
    @layer base {
      * {
        @apply border-border outline-ring/50;
      }

      *:focus-visible {
        outline: 2px solid var(--df-accent-focus);
        outline-offset: 2px;
      }

      /* フォールバック（古いブラウザ向け） */
      @supports not selector(:focus-visible) {
        *:focus {
          outline: 2px solid var(--df-accent-focus);
          outline-offset: 2px;
        }
      }

      body {
        @apply bg-background text-foreground;
      }
    }
    ```

    注意点:
    - `*:focus-visible` はマウスクリック時は非表示、キーボード操作時のみ表示
    - `--df-accent-focus` は既に #3b82f6 で定義済み
    - outline-offset: 2px で要素から少し離して視認性を向上
