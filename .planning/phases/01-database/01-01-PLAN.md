    ```typescript
      id: uuid("id").primaryKey().defaultRandom(),
      title: text("title", { length: 32 }).notNull(),
      artist: text("artist", { length: 32 }),
      color: text("color"), // hex形式（例: #3b82f6）
      filename: text("filename", { length: 32 }).notNull().unique(), // R2キー（例: bgm/lofi-01.mp3）
      tier: text("tier", { enum: ["free", "premium"] }).notNull().default("free"),
      createdAt: timestamp("created_at").notNull().defaultNow(),
      updatedAt: timestamp("updated_at").notNull().defaultNow(),
    });
    ```

    - `filename` に `.unique()` 制約を追加
