## Physical Interaction Contract

### Touch Target Size（TOUCH-01）

全ての小さなアイコンボタンに`p-3`（12pxパディング）を追加して44px以上のタッチターゲットを確保する。

**対象コンポーネント:**
- BgmPlayer: 次/前ボタン（SkipBack/SkipForward）、プレイリストボタン（List）
- TodoItem: 削除ボタン（Trash2）
- TrackItem: 編集ボタン（Edit）、削除ボタン（Trash2）

**実装パターン:**
```tsx
<motion.button
  onClick={handlePrevTrack}
  className="p-3 text-cf-text hover:text-cf-primary hover:bg-white/10 transition-colors"
  aria-label="前の曲"
>
  <SkipBack className="w-6 h-6" />
</motion.button>
```

**例外:**
- 音量スライダーのつまみ（16px）— スライダーは特別なUIパターンとして除外

### Cursor Pointer（TOUCH-02）

`src/index.css`の`@layer base`にグローバル定義を追加する。

**実装パターン:**
```css
@layer base {
  *:focus-visible {
    outline: 2px solid var(--df-accent-focus);
    outline-offset: 2px;
  }

  /* Phase 12追加: グローバルcursor定義 */
  button {
    cursor: pointer;
  }

  /* ドラッグハンドルはTailwindクラスで上書き（cursor-grab active:cursor-grabbing） */

  body {
    @apply bg-background text-foreground;
  }
}
```

### Mobile Overflow（RESP-06）

各カードルートの`overflow-hidden`を削除する。

**対象コンポーネント:**
- TodoList
- CurrentTaskCard
- BgmPlayer

**実装パターン:**
```tsx
// Before
<div className="bento-card flex flex-col min-h-64 sm:h-full overflow-hidden p-4">

// After
<div className="bento-card flex flex-col min-h-64 sm:h-full p-4">
```

**維持:**
- アニメーション用の`overflow-hidden`（Framer Motion等）は維持
- App.tsxのルート要素（`min-h-screen`）の`overflow-hidden`は全画面レイアウト用として維持

### Album Art Size（RESP-07）

BgmPlayerのアルバムアートサイズを96pxで固定する。

**実装パターン:**
```tsx
// Before: レスポンシブサイズ
<div className="relative w-24 h-24 sm:w-32 sm:h-32 flex-shrink-0">

// After: 固定サイズ（96px）
<div className="relative w-24 h-24 flex-shrink-0">
```

## Visual State Transitions

### Hover State（タッチターゲット追加ボタン）

```tsx
className="p-3 text-cf-text hover:text-cf-primary hover:bg-white/10 transition-colors"
```

- ホバー時: `text-cf-primary`（--df-accent-primary）+ `bg-white/10`
